Esta documentación proporciona información detallada sobre el frontend del Sistema Deportivo, desarrollado con Next.js y React. Aquí encontrarás guías de instalación, estructura del proyecto, componentes principales y cómo se integra con el backend.

## Tecnologías Principales
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Icons

## Requisitos del Sistema
- Node.js 18.17 o superior
- npm o yarn
